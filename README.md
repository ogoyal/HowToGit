# HowToGit
How to use Git

1. To know which branch you are on:
   ```
   git branch
   ```

2. To add files to repository:
   ```
   git add file1
   ```

3. Check status of files:
   ```
   git status
   ```

4. See what changed:
   ```
   git diff file1
   ```

5. Commit a file:
   ```
   git commit -m "first commit"
   ```

6. Push file to current branch in repository:
   ```
   git push
   ```
